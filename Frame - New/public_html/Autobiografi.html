Autobiografi

Nama saya Ryan Andyca Rachman, saya lahir di Bandung, 02 Mei 1996, saya anak pertama dari dua bersaudara, Ayah saya Asep Rachman dan Ibu saya Noneng Kusmayati. Ayah saya seorang Karyawan Swasta dan Ibu saya seorang Ibu Rumah Tangga berikut Guru Paud (Pendidikan Usia Dini). Adik saya perempuan yang bernama Nindytha Andryana Rachman, duduk di bangku kelas 2 SMK dan bersekolah di SMK PI Inhoftank, Bandung.

Pada saat umur 5 tahun saya memulai karir pendidikan di TK Al-Amanah, yang berada di Jl. Sukabirus, saya selesai pada tahun ajaran 2001/2002. Saya melanjutkan ke jenjang SD yaitu di SDN 4 Dayeuhkolot, selama 6 tahun saya menuntut ilmu disini hingga lulus pada tahun ajaran 2008/2009 dan mendapatkan ijazah. Kemudian saya melanjutkan ke jenjang selanjutnya yaitu di SMP, saat SMP Saya bersekolah di SMP Negeri 3 Bandung, salah satu SMP favorit di Kota Bandung, berjarak Â± 9 Km dari tempat tinggal saya. Di SMP saya menimba ilmu selama 3 tahun lamanya, setelah 3 tahun tepatnya pada 2010/2011 saya menyelesaikan pendidikan di SMP dan saya mendapatkan ijazah SMP. Kemudian dilanjutkan ke jenjang berikutnya yaitu di SMK MedikaCom, yang berada di Jl. Soekarno-Hatta, disitulah saya mengenakan seragam putih abu-abu dan menuntut ilmu selama 3 tahun. Hingga lulus dan mendapat ijazah SMK pada 2012/2013.

Saya melanjutkan menuntut ilmu di perkuliahan Universitas STTB (Sekolah Tinggi Teknologi Bandung). Saya daftar sebagai mahasiswa Weekend, dimana jam perkuliahan hanya dua hari dalam seminggu, yaitu Sabtu dan Minggu. Sekarang saya baru menyelesaikan semester 3 dan baru masuk ke semester 4. Saya mengikuti perkuliahan Weekend karena saya ingin berkarir juga, jadi selain mengisi waktu dengan kuliah saya juga bisa bekerja. Selang 3 bulan setelah saya masuk kuliah, saya diterima untuk bekerja di salah satu perusahaan swasta di Kota Bandung, yaitu di CV Fajar Photocopy Bandung, distributor Mesin Photocopy yang mencakup penjualan mesin + sparepart, sewa mesin photocopy, juga service mesin photocopy untuk customer.

Saya juga membuat sebuah project band yang hanya terdiri dari dua personil, saya sebagai vokalis dan teman saya sebagai gitaris, project ini saya namakan Last Appetizer, sampai saat ini Last Appetizer masih produksi lagu yang direkam di Home Studio pribadi (LA Home Studio), mengingat hanya memiliki 2 personil, maka band ini memanfaatkan virtual music tools untuk mengaransemen lagu agar terdengar seperti musik full band. Untuk kedepannya saya berharap project ini bisa sukses dan karya-karyanya bisa laris dan terkenal, kalaupun bukan band ini yangsukses, paling tidak LA Home Studio bisa dimanfaatkan sebagai sarana usaha Studio Record untuk umum.
